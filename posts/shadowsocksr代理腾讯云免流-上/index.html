安装配置putty

1.下载

进入 下载地址。 下载 The latest release version (稳定版)的 Zip file: putty.zip。

2.生成公钥和私钥

在 puttygen 里生成公钥保存私钥，并且在 腾讯云控制台 创建刚刚在 puttygen 里生成的公钥。

3.配置

在 putty 中， Window -> Translation -> Remote character set:选择为 UTF-8。

Connection -> SSH -> Auth -> Private key file for authentication: 选择保存好的私钥文件路径。

Session -> Host Name 中编辑 root@你的主机IP (公网IP) -> Saved Sessions 编辑一个名字方便存储 -> save ，下次登录可直接 load。

到此为止完成 putty 的配置， 点击 open 弹出黑框框，此时可开始配置 shadowsocks。
